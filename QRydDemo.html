The Hadamard gate creates an equal superposition of a computational basis state.
The T gate multiplies the basis state one by the complex phase pi/4.
The adjoint T gate multiplies the basis state one by the complex phase -pi/4.
The controlled-NOT gate applies the Pauli-X gate to the target qubit if the control qubit is in the state one.
The controlled-Z gate applies the Pauli-Z gate to the target qubit if the control qubit is in the state one.
The SWAP gate swaps two qubits.
The Pauli-X gate flips a computational basis state.
The Pauli-Y flips a computational basis state and multiplies the result by plus/minus the imaginary unit.
The Pauli-Z gate multiplies the basis state one by minus one.
The square root of the Pauli-X gate.
The square root of the Pauli-Z gate.
The controlled-controlled-NOT gate applies the Pauli-X gate to the target qubit if both control qubits are in the state one.
The controlled-controlled-Z gate applies the Pauli-Z gate to the target qubit if both control qubits are in the state one.
The controlled-SWAP gate swaps two qubits if the control qubit is in the state one.
The RX gate rotates the state of a qubit about the X axis by a user-defined angle.
The RY gate rotates the state of a qubit about the Y axis by a user-defined angle.
The RZ gate rotates the state of a qubit about the Z axis by a user-defined angle.
The phase gate multiplies the basis state one by a user-defined phase.
The controlled-P gate applies the phase gate to the target qubit if the control qubit is in the state one.
The controlled-controlled-P gate applies the phase gate to the target qubit if both control qubits are in the state one.
The identity gate introduces a delay, but does not change the state of an ideal qubit.
